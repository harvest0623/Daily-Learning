## HTML
1. 语义化标签 -- 可读性强，便于搜索引擎的查找
2. Dom 嵌套不要过深
3. script 标签上的 async 和 defer 合理使用

## CSS
1. 选择器简约高效

## JS
1. 事件委托 -- 利用事件冒泡机制，将事件处理函数绑定在父元素上，从而减少事件处理函数的数量
2. 防抖节流
3. 懒加载
4. 虚拟列表
5. webWorker -- 利用多线程机制，将耗时的操作放到子线程中执行，从而避免阻塞主线程

## React
1. useMemo -- 缓存计算结果，避免重复计算
2. useCallback -- 缓存函数，避免重复创建
3. memo -- 缓存组件，避免重复渲染
4. 组件化 -- 将页面拆分成多个组件，每个组件负责自己的状态和渲染，从而提高代码的可维护性和可复用性
