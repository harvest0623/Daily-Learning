## IP定位
- 比如：https://www.baidu.com
- 浏览器得知道 www.baidu.com 对应的 ip 是多少，才能请求到对应的服务器。

## DNS解析
- 浏览器向本地域名服务器查询 ip 地址，如果找不到
- 本地域名服务器 向 根域名服务器 请求ip 地址，如果找不到
- 本地域名服务器 向 顶级域名服务器 请求ip 地址，如果找不到
- 本地域名服务器 向 目标域名服务器 请求ip 地址
- 将 ip 写入 DNS 缓存

**例如：**
> 浏览器会先向 DNS 服务器发送 www.baidu.com 对应的 ip 地址的请求
>
> DNS 服务器会返回 www.baidu.com 对应的 ip 地址
>
> 浏览器会使用这个 ip 地址，向对应的服务器发送请求
>
> 服务器会返回对应的 html 文件
>
> 浏览器会解析这个 html 文件，渲染到页面上
