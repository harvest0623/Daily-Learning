## 函数调用（Function Calling）
1. LLM 只能回答训练截止前的知识，一些实时的、当下的知识，LLM 无法回答
2. LLM 通信时需要接收一个 tools 参数，我们可以指明告诉 LLM，遇到解决不了的问题时，可以调用 tools 中的函数
3. functioncall 机制, 工具调用没有统一的规范，（工具命名，参数传递，返回值等都没有统一的规范）
4. 各家大模型的 functioncall 机制都有自己的实现，（如 OpenAI 的 functioncall 机制，Google 的 functioncall 机制等）

## 什么是大模型函数调用？
大模型函数调用是指在大型语言模型（LLM）中，通过调用外部函数来扩展模型能力的一种技术。它允许模型在处理任务时，根据需要调用特定的函数来获取信息、执行操作或解决问题，从而弥补模型自身知识的局限性或处理复杂任务的能力
