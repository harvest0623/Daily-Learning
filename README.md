# 🌟 Daily Learning

> 一份系统化的前端与AI全栈学习实践库，记录从基础到进阶的技术成长轨迹。

## 📚项目简介

**Daily-Learning** 是一个持续迭代的**个人学习与实践仓库**，专注于**前端开发、全栈技术及算法思维**的沉淀与分享。这里不仅是代码的集合，更是一份可参考的**前端工程师成长路线图**，帮助更多开发者从入门到进阶，构建系统化的技术知识体系。

-   **学习路径清晰**：从 `HTML/CSS` 基础布局、`JavaScript` 核心原理，到 `React`、`TypeScript`、`Node.js（Koa）`等主流框架与服务端开发，再到设计模式、数据结构与算法、性能优化（防抖、懒加载）等进阶主题，形成了完整的技术成长路线。
-   **实战驱动学习**：每个目录都有可运行的实践项目或代码示例，让理论知识落地为可复用的工程能力。
-   **持续迭代与开源共享**：累计 `180+` 次提交，不断补充面试考点、AI 接口开发 `（coze-api）`等前沿内容，**欢迎 ⭐ `Star`、`Fork` 或提交 `Issue`**，一起交流学习、共同进步。

## 📁 项目结构

```plaintext
Daily-Learning/
├── AI/                  # AI 接口开发实践 (玩转 coze、deepseek、可灵等 AI 大模型)
├── CSS/                 # CSS 布局与样式实践 (弹性布局、定位布局、浮动布局)
├── HTML/                # HTML 页面开发 (各种有趣切页面，比如music-store)
├── JS/                  # JavaScript 核心应用 (闭包、预编译、set-map 等各种 JavaScript 知识)
├── JS书写系列/           # JavaScript 性能优化 (防抖节流等)
├── Node/                # Node.js 服务端开发 (koa 等)
├── React/               # React 框架实践 (包含 React 基础知识以及用 React 框架完成的项目)
├── TS/                  # TypeScript 基础实践 (TS 基础知识大全)
├── 数据结构/             # 数据结构与算法 (链表、递归、排序、去重问题等)
├── 设计模式/             # 设计模式实践 (单例模式等)
└── 面试考点/             # 前端性能优化 (懒加载等)
```

## 🚀 快速开始

1.  **克隆项目到本地**

```bash
git clone https://github.com/harvest0623/Daily-Learning.git
```

2.  **进入对应目录**

```bash
cd Daily-Learning/React  # 比如进入 React 文件夹
```

3.  **安装依赖并运行**

```
npm install
npm start
```

> 不同目录下的项目可能有不同的运行方式，请参考各子目录下的 `README.md` 或 `package.json` 脚本。
