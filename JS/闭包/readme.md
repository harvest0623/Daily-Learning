# 作用域
- 在程序中定义变量的区域，该位置决定了变量的生命周期。通俗来讲，作用域就是变量和函数的课访问的范围，即作用域控制着变量和函数的可见性和生命周期

1. 全局作用
2. 函数作用

作用域只能由内到外访问

3. 块级作用域


# 作用域链
v8 在当前作用域中查找一个变量，如果找不到，就会去上一级作用域中查找，再找不到，继续去上一级，层层往上，这种查找关系，就叫作用域链

一个函数被编译时，一定会用一个 outer 指针来记录该作用域的外层作用域的外层作用域是谁

词法作用域：函数声明的位置


# 闭包
1. 一个函数执行完毕后，它的执行上下文会从调用栈中被销毁
2. 一个函数的内部函数一定有权利访问该外部函数中的变量 (作用域的规则)

当调用一个外部函数中返回的内部函数后，即使外部函数已经执行结束，但是内部函数依然引用了外部函数中的变量，那么外部函数的执行上下文就不能被完全销毁，而是会保留一个集合，用来装内部函数需要引用的变量，我们把该集合称之为 ”闭包“

- 优点：定义私有变量，封装模块
- 缺点：内存泄露
